# Python Projects

A collection of Python projects built from scratch while learning programming — starting from zero and progressing through CLI tools, web development, and AI integration.

## Projects

| Project | Description |
|---|---|
| To-Do List | CLI task manager with add, complete, and delete |
| Password Generator | Generates secure random passwords |
| File Organizer | Automatically sorts files into folders by type |
| Expense Tracker | Track and summarize personal expenses |
| Web Scraper | Scrapes data from websites using BeautifulSoup |
| Weather App | Fetches live weather data from an API |
| PDF Merger | Combines multiple PDF files into one |
| Bulk File Renamer | Renames multiple files at once with custom patterns |
| Desktop Reminder App | Sends desktop notifications at scheduled times |
